# gurobi (stub)

This is a stub R package for the [Gurobi Optimizer](https://www.gurobi.com/) R interface.

It exists so that packages listing `gurobi` in `Enhances` (e.g., [CVXR](https://cvxr.rbind.io)) can pass CRAN checks via [r-universe](https://r-universe.dev).

## Installing the real package

The full `gurobi` R package is distributed by Gurobi Optimization and requires a Gurobi license. See the [installation instructions](https://www.gurobi.com/documentation/current/refman/r_ins_the_r_package.html).
